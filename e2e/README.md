# E2E тесты для калькулятора

Этот каталог содержит End-to-End тесты для калькулятора, написанные с использованием Playwright.

## Установка зависимостей

Перед запуском тестов необходимо установить зависимости:

```powershell
npm install
```

После установки зависимостей нужно установить браузеры для Playwright:

```powershell
npx playwright install
```

## Запуск тестов

### Запуск всех E2E-тестов

```powershell
npm run test:e2e
```

### Запуск тестов с UI режимом (интерактивный режим)

```powershell
npm run test:e2e:ui
```

### Запуск тестов в видимом режиме браузера

```powershell
npm run test:e2e:headed
```

### Запуск тестов в конкретном браузере

```powershell
npx playwright test --project=chromium
npx playwright test --project=firefox
npx playwright test --project=webkit
```

## Покрытие тестами

E2E-тесты покрывают следующие сценарии:

- ✅ Начальное состояние калькулятора
- ✅ Ввод чисел (отдельные цифры, множественные цифры, ноль)
- ✅ Добавление десятичной точки
- ✅ Операции (сложение, вычитание, умножение, деление)
- ✅ Вычисления (простые и сложные выражения)
- ✅ Деление на ноль (положительные и отрицательные сценарии)
- ✅ Очистка дисплея
- ✅ Удаление последнего символа
- ✅ Продолжение вычислений после результата
- ✅ Интеграционные сценарии

## Структура тестов

Тесты организованы по функциональным группам:

- `Начальное состояние` - проверка начального состояния калькулятора
- `Ввод чисел` - тесты для ввода цифр
- `Десятичная точка` - тесты для работы с десятичными числами
- `Операции` - тесты для операторов
- `Вычисления` - тесты для выполнения вычислений
- `Деление на ноль` - специальные тесты для обработки ошибок
- `Очистка дисплея` - тесты для функции очистки
- `Удаление последнего символа` - тесты для функции удаления
- `Продолжение вычислений после результата` - тесты для цепочки вычислений
- `Интеграционные сценарии` - комплексные тесты

## Отчеты

После запуска тестов HTML отчет будет доступен по команде:

```powershell
npx playwright show-report
```
